# Web Application Demo

Welcome to the Demo of the Web Apps Design/Demo.

This repository is an experimental space for exploring, prototyping, and validating ideas for modern Progressive Web Applications (PWAs). It serves as a design sandbox where new UI patterns, user experiences, architectures, and web technologies can be tested before being incorporated into production-ready applications.

## Status

This is a early stage demo of the [Web App Manifest](https://w3c.github.io/manifest) spec.
