# Web Applications

Welcome to the home of the Leo's Web Apps project.

# Context & History

## Web Application Prototyping Sandbox (Archived)

This repository originated as an experimental sandbox created by the primary authors of the [Web App Manifest](https://w3c.github.io/manifest) specification. It served as the foundational testing ground where we prototyped design patterns, validated technical feasibility, and stress-tested edge cases to prove the specification's viability before it was widely adopted by modern web browsers.

## Repository Status

With the Web App Manifest specification now fully integrated into standard browser ecosystems, this repository has reached the end of its original lifecycle and is officially archived.

While the project is retired in its original capacity, it has been repurposed by one of the contributors to serve as a base for developing new open-source applications. The `legacy/` directory has been preserved in its entirety to honor the early experimental work and maintain a record of the original implementation.
